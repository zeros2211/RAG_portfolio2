RAG Chat System COMPLETE ✅

A production-ready Retrieval-Augmented Generation (RAG) chat system that enables users to upload PDF documents and ask questions through an intelligent conversational interface.

Quick Start

# 1. Start all services docker-compose up --build # 2. Pull Ollama models (in new terminal) docker exec -it rag_ollama ollama pull qwen3-embedding:0.6b docker exec -it rag_ollama ollama pull qwen3:8b # 3. Access the application open http://localhost

Key Features

📄 PDF Upload
Drag-and-drop interface with automatic processing
🔍 Semantic Search
Vector embeddings with ChromaDB
💬 Streaming Chat
Real-time responses via SSE
📚 Source Citations
Clickable references to source documents
📖 PDF Viewer
Built-in viewer with page navigation
🐳 Docker Deploy
One-command deployment

Technology Stack

  • Backend: Python 3.9+, FastAPI, PyMuPDF, ChromaDB, SQLModel
  • Frontend: React 18, TypeScript, Vite, shadcn/ui, TailwindCSS
  • AI/ML: Ollama (qwen3-embedding:0.6b, qwen3:8b)
  • Infrastructure: Docker, Docker Compose, Nginx

Project Structure

  • backend/ - FastAPI application with RAG pipeline
  • frontend/ - React TypeScript application
  • data/ - Persistent storage (PDFs, vectors, sessions)
  • docker-compose.yml - Service orchestration
  • verify-setup.sh - Setup verification script
